## Features

- Add, update, and remove books
- Manage library members
- Issue and return books with due-date tracking
- Search books by title, author, or ISBN
- Basic validation and error handling

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Kerry-Mannette/Library-Management-System-Application.git
   cd Library-Management-System-Application
   ```

2. Install dependencies / build
   - Java / Maven example:
     ```bash
     mvn clean install
     ```
   - Node example:
     ```bash
     npm install
     ```

3. Configure environment
   - Copy example env file and update database settings / secrets:
     ```bash
     cp .env.example .env
     # edit .env
     ```

### Run

- Java example:
  ```bash
  mvn spring-boot:run
  ```
- Node example:
  ```bash
  npm start
  ```

Open your browser at http://localhost:8080 (or the configured port).
